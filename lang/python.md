### コレクション
python では リスト、タプル、辞書、集合のこと。

	# リスト。多言語で言うところの配列。
	>>> t = [1,2,3,4]

	# タプル。読み取り専用コレクション。
	>>> t = (1,2,3,4)

	# アクセス方法はどちらも同様
	>>> t[0] + t[1]
	3
	
	# コレクションの操作
	# 追加
	>>> t.append(4)
	>>> t
	[0,1,2,3,4]
	
	# 挿入
	>>> t.insert(0,-1)
	>>> t
	[-1,0,1,2,3,4]
	
	# 削除
	# del キーワードを使用しているが pop() もある様なので使い分ける。
	>>> del t[0]
	>>> t
	[0,1,2,3,4]

## 制御構文

### if

	# 条件式に使える記号に関しては他の言語とほぼ同様。
	# 複数条件の指定に関しては and or not を使用。
	# 条件式と else の後ろには : が付く。
	# インデントによってブロックを形成する。

	>>> A = 3
	>>> B = 5
	>>> if A < 10 and B < 10 :
	... 	print("less than 10")
	...
	less than 10
	
	# not の場合は if の直後に記入。
	# else の行ではインデントは不要。
	>>> if not A == 4 :
	... 	print("not 4")
	... else :
	... 	print("4");
	...
	not 4
	
	# else if は elif と記述。
	>>> if A == 4 :
	... 	print("A is 4")
	... elif A == 2 :
	... 	print("A is 2")
	... else : 
	... 	print("A is ?")
	... A is ?
	...
	A is ?
	
	# 尚、インデントを使用せずに１行で書く方法も可能らしい。あまり使わない方向で行きたい。
	>>> if not A == 4 : print("not 4");
	...
	not 4

備考 : 条件式に対して 0、空文字、空のリスト、空のタプルを入力した場合は False で判定される。

### 三項演算子

	# x に対して、a > 0 ならば 10 それ以外なら 20 を代入する。
	>>> x = 10 if a > 0 else 20

### while

	# 指定した条件が真の間繰り返す。普通の while
	>>> while counter < 3 :
	... 	print(counter)
	... 	counter += 1
	...
	0
	1
	2
